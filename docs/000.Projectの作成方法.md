# プロジェクトの作成方法

## 1. ソリューション作成
```Bash
mkdir SelfCSharp
cd SelfCSharp
dotnet new sln -n SelfSCharp
```

## 2. Chap01(親フォルダ)を作成
```Bash
mkdir Chap01
```

## 3. 子プロジェクトの作成
```Bash
dotnet new console -o Chap01/TopLeval
```

## 4. ソリューションにプロジェクトを追加
```Bash
dotnet sln add Chap01/TopLeval/TopLevel.csproj
```

## 5. 実行方法
```Bash
dotnet run --project Chap01/TopLeval
```
